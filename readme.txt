A widget that displays a random Quranic verse

== Description ==

A widget that displays a random Quranic verse. You can select from 
72 different languages for the Quranic verse. It also gives a link at the end 
of the verse to iknowledge.islamicnature.com where you can read and listen to 
the rest of the Surah, read the tafsir of the Surah and download the mp3 of the 
verse from different recitors.
Tags: koran, quran, muslim, islam, plugin, verse, widget.

== Installation ==

1. Extract the folder into the wp-content/plugins directory.
2. Go to the plugins section in the admin backend and activate the widget.
3. Go to Appearance > Widgets and add the widget to your sidebar.
4. Enjoy.
